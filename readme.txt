Powerful block to create custom designs and patterns library.

== FOR USERS ==

The collection of patterns with creative modern describes. 

Simple editing mode, to allow you edit the content on the fly.

== FOR DESIGNERS ==

Powerful tools to create custom designs. 

== FEATURES ==

1. Move the items where you want
2. Custom shape figures generator
3. Custom image & background masks
4. Custom shadows & shadow patterns
